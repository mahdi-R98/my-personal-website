نمونه پروژه‌ها

وبسایت نمونه - طراحی واکنش‌گرا +

این پروژه شامل طراحی یک وبسایت واکنش‌گرا است که بر اساس تکنولوژی‌های HTML5، CSS3 و جاوااسکریپت ساخته شده است. تمرکز بر UX و طراحی مینیمال می‌باشد.

اپلیکیشن مدیریت تسک با React +

این اپلیکیشن با React و Redux پیاده‌سازی شده و امکان افزودن، ویرایش و حذف تسک‌ها را دارد. همچنین قابلیت فیلتر تسک‌ها بر اساس وضعیت انجام شده یا نشده فراهم است.

سایت فروشگاهی ساده با Node.js و Express +

در این پروژه از Node.js و Express برای پیاده‌سازی یک سایت فروشگاهی ساده استفاده شده است که شامل بخش مدیریت محصولات، سبد خرید و پرداخت است.
