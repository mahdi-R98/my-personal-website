درباره من

سلام! من مهدی هستم، توسعه‌دهنده وب با بیش از ۵ سال تجربه در طراحی و پیاده‌سازی سایت‌های حرفه‌ای و مدرن. عاشق کدنویسی و خلق تجربه‌های جذاب برای کاربران هستم.

در طول سال‌ها، پروژه‌های متنوعی را انجام داده‌ام که شامل سایت‌های شخصی، فروشگاهی، و سیستم‌های مدیریت محتوا بوده است.

مهارت‌ها

فرانت‌اند

  • HTML5، CSS3، JavaScript (ES6+)
  • فریم‌ورک‌های React.js و Vue.js
  • طراحی واکنش‌گرا (Responsive Design) با Bootstrap و Flexbox

بک‌اند

  • PHP و CodeIgniter
  • Node.js و Express.js
  • کار با پایگاه داده MySQL و MariaDB

ابزارها

  • Git و GitHub
  • Docker و محیط‌های مجازی‌سازی
  • Visual Studio Code و Android Studio

پروژه‌ها

سیستم مدیریت کتابخانه

یک سامانه تحت وب برای مدیریت امانت و برگشت کتاب‌ها به صورت آنلاین، با استفاده از PHP و MySQL.

اپلیکیشن اندروید آموزش زبان

اپلیکیشن چندزبانه برای یادگیری زبان با تمرکز بر گرامر و لغات، توسعه داده شده با Kotlin و Android Studio.

وبسایت فروشگاهی

وبسایت فروشگاه اینترنتی با امکانات سبد خرید، پرداخت آنلاین و پنل مدیریت، با React و Node.js.
